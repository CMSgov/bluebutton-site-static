DMERC Line Pricing State Code (SSA)

The 2-digit SSA state code where the durable medical equipment (DME) supplier was located; used by the Medicare Administrative Contractor (MAC) for pricing the service. Source: https://bluebutton.cms.gov/resources/variables/dmerclineprcngstatecd
